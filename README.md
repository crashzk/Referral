# Referral
У каждого игрока есть своя реферальная ссылка и короткий код. Новый человек вводит код при регистрации или переходит по ссылке — дальше начисления идут по вашим правилам: сколько получает пригласивший, есть ли бонус новичку, через сколько дней активности платить рефереру, сколько приглашений максимум на одного человека. Самореферал можно запретить.
